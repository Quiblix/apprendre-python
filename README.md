# Apprendre Python - Collection de Projets

Ce dépôt contient des exercices pratiques conçus pour apprendre les bases de la programmation en Python, tels que la gestion des entrées utilisateur, les structures conditionnelles, les boucles et l'utilisation de modules standards.

## Projets inclus

### 1. Nombre Mystère
Un jeu interactif où l'utilisateur doit deviner un nombre généré aléatoirement.
- **Emplacement** : `nombre mystere/main.py`
- **Fonctionnalités** :
    - Génération d'un nombre aléatoire entre 1 et 100.
    - Calcul et affichage de l'écart (distance) entre la proposition et la cible.
    - Compteur de tentatives pour suivre votre progression.

### 2. Calculatrice
Une calculatrice simple capable d'effectuer les opérations arithmétiques de base via la console.
- **Emplacement** : `calculatrice/main.py`
- **Opérations supportées** : Addition (+), Soustraction (-), Multiplication (*) et Division (/).
- **Gestion d'erreurs** : Le script détecte les tentatives de division par zéro et demande une nouvelle saisie le cas échéant.

## Installation et Utilisation

1. Assurez-vous d'avoir Python 3 installé sur votre système.
2. Naviguez dans le répertoire du projet de votre choix.
3. Lancez le script avec la commande suivante :

```bash
python "nom_du_projet/main.py"
```
